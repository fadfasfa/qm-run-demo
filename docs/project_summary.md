# 项目说明

这个 demo 展示的是一个本地运行的《英雄联盟》伴生查询工具的公开版：

- 英雄数据整理
- 海克斯推荐与协同查询
- 本地 Web 页面
- 桌面浮窗伴生界面

## 做了哪些裁剪

- 移除了本地缓存和运行产物
- 移除了图标、数据和日志类大文件
- 移除了工作区元文件和无关子项目
- 保留了核心源码、`static/` 前端资源和依赖说明

## 哪些内容不在公开版里

- 已生成的 `config/` 数据
- 已下载的 `assets/` 图标和图片
- `logs/`、`temp/`、`tmp/`、`build/`、`dist/`
- 本地编辑器和自动化工作区配置

## 运行限制

- 公开版更适合阅读和演示，不包含完整的私有缓存数据
- 某些页面在首次启动时可能需要联网补全数据
- 桌面浮窗相关能力主要面向 Windows
